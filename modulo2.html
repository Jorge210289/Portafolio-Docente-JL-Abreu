Módulo 2: Estrategias y Herramientas de Enseñanza

Este módulo evidencia mi dominio de las **estrategias pedagógicas** y la integración efectiva de las **Tecnologías de la Información y la Comunicación (TIC)** para promover un aprendizaje activo y colaborativo, esenciales en la formación técnica profesional.

2A. Plataforma de Colaboración y Comunicación

Uso de Moodle para el Aprendizaje Asincrónico y Colaborativo

Utilizo la plataforma **Moodle** como entorno virtual de aprendizaje (EVA) para extender el aula, gestionar recursos y fomentar la discusión. Esta herramienta es fundamental para las asignaturas técnicas que requieren revisión constante de código y conceptos.

**Actividad Evidenciada:** **Foro de debate sobre seguridad en bases de datos** (Asignatura: Diseño y Desarrollo de Base de Datos). A través de este foro, los estudiantes analizan escenarios de vulnerabilidad y proponen medidas de mitigación, aplicando su comprensión de la **Gestión de Riesgos** (ligado a mi formación).

2B. Recursos Digitales para la Práctica Técnica

Para el desarrollo de competencias prácticas en las asignaturas de diseño, desarrollo y mantenimiento, se hace uso constante de herramientas profesionales que simulan el entorno laboral:

  • **Python:** Usado en **Diseño y Desarrollo de Aplicaciones Informáticas** como lenguaje introductorio a la programación, por su sintaxis clara y su aplicación en scripting y automatización de sistemas.
  • **Visual Studio Code (VS Code):** Empleado como editor de código universal para **Portales Web y Recursos Multimedia**, facilitando la depuración y la colaboración en proyectos con Git/GitHub.
  • **Microsoft Access:** Utilizado en **Taller de Ofimática** y la fase inicial de **Diseño y Desarrollo de Base de Datos** para entender los conceptos de modelado de datos y gestión de información a pequeña escala.
